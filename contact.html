Before You Ask

  • Do you have a Discord server?
    No, not anymore. Discord deleted my server and banned my account. Talking about cheats / hacks is now against their rules. I don't want to make another Discord server because I'm tired of getting banned and loosing all my contacts, messages, etc. every time. Please understand.

  • How do I install Wurst? What do I do with the downloaded files?
    Please read the installation tutorial.

  • I have Minecraft 1.12.2. Why does the Wurst MC 1.12 installer say that it couldn't find my Minecraft installation?
    You need Minecraft 1.12, not 1.12.2. Open the Minecraft launcher, create a profile using "release 1.12" and launch it once. Then the installer will work.

  • Why can't I run the cross-platform Wurst 6 installer? (Minecraft 1.8 - 1.12, file ends in .JAR)
    Maybe you don't have Java installed (or you are using an outdated version). Maybe you have configured your computer to open .JAR files with some other program than Java. Or maybe you are simply using "java.exe" instead of "javaw.exe".

  • Which key opens the GUI? How do change my keybinds? etc.
    Please read the .binds command tutorial.

  • Help! My X-Ray is not working.
    Are you seeing fake ores everywhere? That's a server plugin that's meant to break X-Ray. There is unfortunately nothing you can do about that. Are you seeing everything as if X-Ray was turned off? That's a bug in OptiFine version F4. Please use OptiFine F3 instead.
