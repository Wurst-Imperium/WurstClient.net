## Changelog

- Added [ColorSetting](https://wiki.wurstclient.net/colorsetting): A new type of setting for managing colors, complete with color pickers and support for hex codes. (Thanks to <a href="https://github.com/hiddenalt-team" target="_blank">hiddenalt-team</a>!)

- Added a [`.setcolor`](https://wiki.wurstclient.net/cmd/setcolor) command. Allows you to change ColorSettings through keybinds. (Thanks to <a href="https://github.com/hiddenalt-team" target="_blank">hiddenalt-team</a>!)

- Replaced the sliders for [ClickGUI](https://wiki.wurstclient.net/clickgui)'s background and accent colors with ColorSettings. (Thanks to <a href="https://github.com/hiddenalt-team" target="_blank">hiddenalt-team</a>!)

- Added a "Color" setting to [BaseFinder](https://wiki.wurstclient.net/basefinder). Determines which color BaseFinder uses to highlight man-made blocks.

- Added a "Color" setting to [CaveFinder](https://wiki.wurstclient.net/cavefinder). Determines which color CaveFinder uses to highlight caves.

- Added an "Opacity" setting to [CaveFinder](https://wiki.wurstclient.net/cavefinder). Allows you to disable the default breathing animation and set a fixed level of opacity instead.

- Added a "DD color" setting to [AutoFish](https://wiki.wurstclient.net/autofish). Determines the color of the debug draw (if enabled).

- Added a "Color" setting to [Trajectories](https://wiki.wurstclient.net/trajectories).

- Added a "Color" setting to [ItemESP](https://wiki.wurstclient.net/itemesp).

- Added an "ESP color" setting to [BowAimbot](https://wiki.wurstclient.net/bowaimbot).

- Added "Chest color", "Trap color", "Ender color", "Shulker color" and "Cart color" settings to [ChestESP](https://wiki.wurstclient.net/chestesp).

- Added a "Tracer color" setting to [Freecam](https://wiki.wurstclient.net/freecam).

- Added "Sort by" and "Reverse sorting" settings to [HackList](https://wiki.wurstclient.net/hacklist). Allows you to customize how hacks in the HackList are sorted.

- Added a "Potion type" setting to [TrollPotion](https://wiki.wurstclient.net/trollpotion). Allows you to choose whether to generate a splash potion, a regular potion, a lingering potion or a tipped arrow.

- Added a "Potion type" setting to [KillPotion](https://wiki.wurstclient.net/killpotion). Allows you to choose whether to generate a splash potion, a regular potion or a lingering potion (arrows don't work with this effect).

- Removed [ItemESP](https://wiki.wurstclient.net/itemesp)'s broken "Show item names" setting.

- Fixed a crash when turning off [TreeBot](https://wiki.wurstclient.net/treebot) before it manages to find a tree. (Thanks to <a href="https://github.com/xuri02" target="_blank">xuri02</a>!)

- Fixed a crash when [BaseFinder](https://wiki.wurstclient.net/basefinder) is enabled on startup.

- Fixed [HackList](https://wiki.wurstclient.net/hacklist) glitching out for a split-second when opening ClickGUI.

- Fixed settings windows in [ClickGUI](https://wiki.wurstclient.net/clickgui) sometimes opening at the wrong position.

## Installation

See [this tutorial](/tutorials/how-to-install/wurst-7/) for a more detailed explanation with pictures.

1. **Download the <a href="https://fabricmc.net/use/installer/" target="_blank" rel="nofollow">Fabric installer</a>** (choose either <kbd>EXE</kbd> or <kbd>JAR</kbd>)

1. **Run the Fabric installer** (if you can't run it, <a href="https://www.youtube.com/watch?v=Wv0vPUwitJs" target="_blank">install Java</a> first)

1. **Download Wurst and the Fabric API** (see the [download buttons](#downloads) below).

1. **Place both Wurst and the Fabric API in your `%appdata%/.minecraft/mods` folder.**

If you can't launch the game, it usually means you have the wrong version of something.
See ["How to fix your Wurst installation"](https://www.reddit.com/r/WurstClient/comments/hg6xce/cant_launch_the_game_read_this_first/) for details.
