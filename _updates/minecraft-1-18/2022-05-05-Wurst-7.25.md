## Changelog

- [ChestESP](https://wiki.wurstclient.net/chestesp) can now show barrels again.

- [ChestESP](https://wiki.wurstclient.net/chestesp) can now also show hoppers, droppers, dispensers, hopper minecarts, and furnaces.

- Added [color settings](https://wiki.wurstclient.net/colorsetting) for all of the new types of containers that ChestESP can now show.

- Every type of container that [ChestESP](https://wiki.wurstclient.net/chestesp) can show, except for normal chests, can now be turned off separately with a checkbox.

- Renamed ChestESP's "Trap color" setting to ["Trap chest color"](https://wiki.wurstclient.net/chestesp#trap_chest_color).

- Renamed ChestESP's "Cart color" setting to ["Chest cart color"](https://wiki.wurstclient.net/chestesp#chest_cart_color).

- Moved ChestESP's ["Style" setting](https://wiki.wurstclient.net/chestesp#style) to the top.

- Fixed [ChestESP](https://wiki.wurstclient.net/chestesp) sometimes causing the game to crash.

- Added a "Speed" slider to [Killaura](https://wiki.wurstclient.net/killaura), [KillauraLegit](https://wiki.wurstclient.net/killauralegit), [ClickAura](https://wiki.wurstclient.net/clickaura), [MultiAura](https://wiki.wurstclient.net/multiaura), [TP-Aura](https://wiki.wurstclient.net/tp-aura), [FightBot](https://wiki.wurstclient.net/fightbot), [Protect](https://wiki.wurstclient.net/protect), and [TriggerBot](https://wiki.wurstclient.net/triggerbot) that determines their attack speed. When set to zero, the speed is dynamically adjusted to match your attack cooldown.

- Fixed ModMenu conflicting with the "Wurst Options" button. (Thanks to <a href="https://github.com/Kadeluxe">Kadeluxe</a>!)

- Fixed some of Wurst's screens not behaving correctly when pressing Escape.

- Fixed [ClickAura](https://wiki.wurstclient.net/clickaura) being limited to the cooldown speed even when spam-clicking the attack button.

- The [CaveFinder Opacity](https://wiki.wurstclient.net/cavefinder#opacity) slider now says "breathing" instead of "Breathing" when set to zero.

- Removed [MultiAura](https://wiki.wurstclient.net/multiaura)'s old "Speed" and "Use cooldown" settings, as the new speed slider makes them obsolete.

- On servers with chat preview enabled, [.say](https://wiki-edit.wurstclient.net/cmd/say) will now send spoofed preview data to make it look like a legit message. (22w19a and later versions)

- All other [.commands](https://wiki-edit.wurstclient.net/command#list_of_commands) won't send chat preview data at all, so server admins can't watch you type in your Wurst commands even when previews are enabled. (22w19a and later versions)

## Installation

1. Install <a href="https://fabricmc.net/use/installer/" target="_blank" rel="nofollow">Fabric (Loader)</a>.

2. Add Fabric (API) to your mods folder.

3. Add Wurst to your mods folder.

If you need more details or run into problems, check [this tutorial](/tutorials/how-to-install/wurst-7/).
