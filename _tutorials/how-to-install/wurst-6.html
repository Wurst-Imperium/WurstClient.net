This installation tutorial is only for old Minecraft versions ranging from 1.12.2 - 1.7.2, which came with a Wurst installer. Newer versions work differently. If you want to install Wurst for Minecraft 1.14.2 or newer versions, please follow the Wurst 7 installation guide instead. If you want to install ForgeWurst, please follow the ForgeWurst installation guide instead.

Which Wurst 6 installer do you have?


Wurst installer for Windows (.ZIP)

Cross-platform Wurst installer (.JAR)

Using the Wurst installer for Windows (.ZIP)

  1. Download the Wurst Client, if you haven't already.

    • Always use the latest version that is compatible with your Minecraft version.

  2. Extract the Wurst installer (.ZIP file) that you just downloaded.

    • This will look different depending on what software you use to deal with zip files.
    • Do not simply open the zip file, you really have to extract it!

  3. Find "Wurst Installer.cmd" among the files you just extracted and run it.

    • Again, don't run it from inside the zip file. Use your extracted files.

  4. Click the Install button.

  5. You will normally get a message telling you that Wurst has been successfully installed.

    • If you see an error message instead, and it tells you that the Wurst installer couldn't find your Minecraft installation, simply download that Minecraft version through the launcher and try again.

  6. Open the Minecraft launcher. (Close and re-open the launcher if it was already open.)

  7. Select the new Wurst profile and press PLAY.

Using the cross-platform Wurst installer (.JAR)

  1. Download the Wurst Client, if you haven't already.

    • Always use the latest version that is compatible with your Minecraft version.

  2. Right-click the Wurst installer (.JAR file), select "Open with" and then choose Java.

    • Depending on your system, you will see either "Java(TM) Platform SE binary", "OpenJDK Platform binary", or both.
    • It doesn't matter which Java binary you use, just don't use WinRAR. WinRAR is not Java.
    • If you don't see Java in the list, select "Choose another app".
    • If you can't find Java at all, install Java first.

  3. Click the Install button.

  4. You will normally get a message telling you that Wurst has been successfully installed.

    • If you see an error message instead, and it tells you that the Wurst installer couldn't find your Minecraft installation, simply download that Minecraft version through the launcher and try again.

  5. Open the Minecraft launcher. (Close and re-open the launcher if it was already open.)

  6. Select the new Wurst profile and press PLAY.
